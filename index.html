What is this app?

Senior Design, Team C - This app is a mathematical algorithm of the nueronetwork of how the brain reacts, and equates out how a common English word should be pronounced. On each words, the user must give the best recording of the word to our algorithm hosted on the cloud, and get a score ranging from 0 - 1 (the closer it is to 1, the better the pronounciation was) back. The goal of the app is to help non - native English speakers be able to pronounce common words properly, and our goal is to bring this capability into fruition for you.

Let's cover some assumptions & constraints

Assumptions

  • User is currently using Android smart phone and has basic knowledge to use phone
  • User must be able to speak into the smart phone
  • The system contains volatile storage through SQL
  • The system will be forking a connection into the cloud based SQL server
  • Network issues will be ignored
  • Sound issues will be ignored

Constraints

  • Team lacks professional level android developement skills
  • Team schedule too aggressive
  • Team is not versatile in database management and programming (SQL) skills
