What is this app?

Senior Design, Team C - Our English Language App assists users in improving their pronounciation of various english words. For each word, the user provides a recording of themselves saying that word. They then send this recording to our application server where it is compared against native speakers' recordings. The user recieves a score reflecting this comparison, ranging from 0 - 1 (the closer it is to 1, the better the pronounciation was).

Let's cover some assumptions & constraints

Assumptions

  • User is currently using Android smart phone and has basic knowledge to use phone
  • User must be able to speak into the smart phone
  • The system will be forking a connection into the cloud based SQL server
  • Network issues will be ignored
  • Sound issues will be ignored

Constraints

  • Team lacks professional level android developement skills
  • System not designed to scale up
  • System uses mean aggregations to compare audio clips, loses some granularity
